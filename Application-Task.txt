промпт для создания приложения учета и анализа опросов клиентов
Создать приложение (базу данных) для детского коррекционного центра для учета, анализа и контроля качества клиентских опросов.
🔹 Общая логика системы
По каждому клиенту проводится 3 плановых опроса.
Опросы провидит непосредственно пользователь.
Если выявлено недовольство или недопонимание — создаются дополнительные контакты.
Система должна фиксировать:
Жалобы на сотрудников
Жалобы на условия центра (ремонт, чистота, вода и т.д.)
Система должна позволять анализировать качество работы и управлять конфликтами.
Необходимо выполнить миграцию в приложение из файла microsoft excel Опросы.xlsx с данными по проделанной уже работе.

🔹 Автоматическая логика системы
По каждому клиенту создаются 3 плановых опроса.
Если:
Удовлетворённость = «Не доволен»
ИЛИ
Выявлено недопонимание = «Да»
ИЛИ
Есть любая жалоба (на сотрудника или условия)
→ Статус ситуации автоматически = «В процессе»
→ Разрешается создание дополнительных контактов
→ Увеличивается счётчик контактов
Если заполнено поле «Результат урегулирования» → статус = «Улажена».
Если заполнено поле «Причина неурегулирования» → статус = «Не улажена».
🔹 Необходимая аналитика
Система должна позволять:
% удовлетворённых клиентов
Количество конфликтов «В процессе»
Среднее количество контактов до урегулирования
Количество жалоб по каждому сотруднику
Частые категории жалоб на условия центра
Повторные обращения одного клиента
Самые частые причины недовольства
К аналитике нужно добавить следующие функционал возможность выгрузки всех неудовлетворенных обратных связей в макдаун-файл для последующей обработки нейросетью. 
Также добавить модуль для возможности обработки неудовлетворённых обратных связей через интерфейс приложения при помощи API open роутер с возможностью подключения различных моделей непосредственно через диалоговое окно модуля, c системным промптом для обработки обратной связи. 
🔹 Цель системы
Повышение качества услуг, снижение конфликтов, управляемость процессов, прозрачная аналитика по персоналу и условиям центра.
